# Basic Title Template

A simple template that generates concise session titles.

```handlebars
Generate a short, descriptive title (3-5 words) for this coding session. First
message:
{{firstMessage}}

Output ONLY the title, no quotes or punctuation.
```

**How it works:**

- Uses `firstMessage` variable to capture the user's initial request
- Simple instructions for the model to follow
- Results in titles like "Fix login bug" or "Add user auth"
